## Installation

### Backend Setup

```bash
cd backend
npm install
cp .env.example .env
# Configure your .env file
npm start
```

**Dependencies**: Check `backend/package.json`

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

**Dependencies**: React, Vite, Tailwind CSS, PostCSS
